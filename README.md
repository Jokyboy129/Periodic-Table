# Periodic Table App

An accessible, bilingual periodic table application built with Python and wxPython.  
Designed with a strong focus on accessibility (A11y) and usability for screen reader users.

## Features

- Bilingual support (English / German)
- Fully accessibility-focused (screen reader optimized, A11y)
- Classic list view and interactive grid view
- Search elements by name, symbol, or atomic number
- Filters for element groups (e.g. halogens, lanthanides, transition metals)
- Molar mass calculator for chemical formulas
- Settings for language and accessibility options
- High-contrast and large-font modes for better readability
