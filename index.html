Periodic Table App

An accessible, bilingual periodic table application built with Python and wxPython. It is designed with a strong focus on accessibility and screen reader support.

Features

  • Bilingual support (English / German)
  • Screen reader friendly (A11y optimized)
  • List and grid view for elements
  • Search by name, symbol, or atomic number
  • Element group filters
  • Molar mass calculator
  • High contrast and large font options
